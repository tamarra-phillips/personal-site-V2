Key Points About Me

Research and Analytics

My current passion is UX reasearch. Deep diving into the analytics of a page or a promotional campaign to see what is converting. Setting up heatmaps and session recording to gain a greater understanding of how a user is interacting with a page or email. A/B testing to get feedback from customers in an indirect manner. Using the results that best serves the customer. Data driven decisions are the best decisions.

HTML, CSS, SCSS, Javascript, jQuery with some Angular and a tiny ammount of C#. I have always enjoyed the more visual sides of coding. I like that I can write a little code, save and instantly see the changes that I have made. There is so much more to quality front end code than the languages you are writing in. You have to be mindful of SEO. Pay attention to Meta Tags and Schema when needed. Know your analytics strategy and create items for tracking accordingly.
